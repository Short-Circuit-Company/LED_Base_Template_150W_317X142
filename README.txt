LED Base Template 150W (317 x 142)

Overview
This KiCad template provides a starting point for LED street light base PCBs rated around 150 W with a board outline of approximately 317 mm x 142 mm. It includes preconfigured project files: schematic (`LED_PCB_Street_150w.kicad_sch`), PCB layout (`LED_PCB_Street_150w.kicad_pcb`), and project settings (`LED_PCB_Street_150w.kicad_pro`).

Contents
- Schematic: LED_PCB_Street_150w.kicad_sch
- PCB Layout: LED_PCB_Street_150w.kicad_pcb
- Project: LED_PCB_Street_150w.kicad_pro
- Images: images/ (optional preview assets)
- License: LICENSE.txt
- Metadata: meta/template.json

Requirements
- KiCad 9.0 or newer

Installation
Place this folder under your KiCad user templates directory. On Windows, a typical path is:
%USERPROFILE%\Documents\KiCad\9.0\template\
Ensure this template's folder name remains: LED_Base_Template_150W_317X142

Usage
1. In KiCad, choose File > New > Project from Template.
2. Select the LED Base Template 150W (317 x 142).
3. Choose a destination directory and project name; KiCad will copy these files.
4. Update schematic symbols, footprints, and parameters to match your design needs.
5. Verify design rules (DRC/ERC) and manufacturing constraints before production.

Notes
- Dimensions and power rating are indicative; validate thermal and electrical performance.
- Replace any placeholder assets in `images/` with your own icon/preview if desired.
