LED Base Template 150W (317 x 142)

Short Circuit Company — KiCad 9 template

Overview

Template for an LED street light base PCB (~150 W) with a 317 mm × 142 mm outline. Includes schematic, PCB, and project settings to accelerate design.

KiCad 9 LED Lighting High Power Metric

Specifications

  • Board size: 317 mm × 142 mm
  • Application: Street light base (~150 W)
  • Files: schematic, PCB, project settings

Usage

  • File → New → Project from Template → select this template
  • Adjust symbols, footprints, and nets to your design
  • Run ERC/DRC; verify fab notes per manufacturer
